Olá! Este README existe para dar informações necessárias acerca do site.

O código-base oferecido pelo professor foi utilizado apenas para facilitar a estrutura do HTML e funcionalidades como a troca de imagens de fundo e temporizador. Fora isso, reconstruímos o código do zero para moldá-lo ao tema e intensificar nosso conhecimento nas tecnologias.

Nosso projeto não foi dividido em partes para cada um desenvolver, todos trabalharam em conjunto em todos os arquivos e áreas do projeto, para garantir boa sintonia e autonomia de cada um. Para trabalharmos de forma harmônica, criamos um repositório no GitHub e, com gestão de tempo e comunicação, fomos contribuindo no projeto com commits e lembretes do que foi alterado, para não haver merges ou falhas constantes no projeto.

No arquivo "index.html", aplicamos apenas a estrutura base da página, onde há as divs e referências aos arquivos CSS e JS, para aplicar boas práticas de organização no código.
